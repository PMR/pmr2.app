PMR2 Interfaces
===============

This is the interface to the root repository object.

    >>> import pmr2.app.interfaces
    >>> from pmr2.app.interfaces import IPMR2
    >>> type(IPMR2.get('title'))
    <class 'zope.schema._bootstrapfields.TextLine'>
    >>> type(IPMR2.get('repo_root'))
    <class 'zope.schema._bootstrapfields.TextLine'>

For the add form

    >>> from pmr2.app.interfaces import IPMR2Add
    >>> type(IPMR2Add.get('id'))
    <class 'zope.schema._field.BytesLine'>
